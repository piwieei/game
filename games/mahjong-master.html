Game Introduction

Mahjong Master is a classic Chinese Mahjong game that perfectly combines traditional gameplay with modern interface design. The game follows classical Mahjong rules, requiring players to use strategic thinking and memory to achieve victory. Game features include:

  • Beautiful game interface and animation effects
  • Supports single-player and multiplayer modes
  • Detailed tutorials and game guides for beginners
  • Rich game statistics and achievement system
  • Regularly updated game content and events

Operation Instructions

Basic Operations

  • Use mouse to select Mahjong tiles
  • Click "Chow", "Pung", "Kong" buttons for corresponding actions
  • Click "Win" button to complete the game

Keyboard Shortcuts

  • Space key - Skip current turn
  • Enter key - Confirm action
  • ESC key - Pause game

Game Rules

The game follows traditional Chinese Mahjong rules, including:

  • Standard 136 tiles
  • Supports Chow, Pung, Kong, Win and other basic gameplay
  • Supports self-drawn and discard wins
  • Special combinations: Seven Pairs, Thirteen Orphans, Pure Hand, etc.
